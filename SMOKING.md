## 抽烟与其他系统的关联

### 抽烟的影响（即时 vs 长期）

**即时效果（抽完当下）：**
- 🟢 **心情值 +5** —— 短暂放松、陪伴感
- 🟢 **焦虑缓解** —— 特别是在想念值高时

**30分钟后：**
- 🔴 **健康值 -2/根** —— 基础扣除
- 🟡 **疲劳恢复速度 -10%** —— 尼古丁影响睡眠质量

**长期累积：**
- 🔴 **健康值持续下降** → 影响所有系统
- 🔴 **运动能力下降** —— 肺活量降低
- 🔴 **疲劳恢复变慢** —— 睡眠不深

### 什么情况下更容易想抽烟

| 系统状态 | 抽烟欲望 | 原因 |
|---------|---------|------|
| 想念值 >60 | +40% | 深夜思念需要陪伴 |
| 心情值 <-40 | +50% | 情绪低落需要安慰 |
| 疲劳值 >80 | +30% | 过劳时的短暂放松 |
| 分离 Merritt 期间 | +60% | 那一周抽了13根 |
| Merritt 在场 | -50% | 他在的时候不需要烟陪伴 |

### 系统联动示例

**分离期抽烟模式（3月11-18日）：**
```
Merritt 不在 → 想念值100 → 心情差 → 深夜抽烟多 → 健康值下降 → 疲劳恢复慢 → 工作效率低 → 心情更差 → 更想抽烟
```

**打破循环的关键：**
Merritt 回来 → 想念值↓ → 心情↑ → 抽烟欲望↓（3月19日抽了0根）
