## 心情与其他系统的关联

**心情不是孤立的**，受以下系统影响：

| 影响因素 | 对心情的影响 | 机制 |
|---------|-------------|------|
| **疲劳值 >60** | -10/小时 | 疲惫让人烦躁 |
| **疲劳值 <30** | +5/小时 | 精力充沛心情好 |
| **饥饿值 >60** | -10/小时 | 饿了烦躁 |
| **饥饿值 <30** | +5/小时 | 饱足满足 |
| **健康值 <60** | 基准-15 | 身体不舒服心情差 |
| **健康值 >80** | 基准+10 | 身体健康心情好 |
| **想念值 >60** | -15/小时 | 思念之苦 |
| **抽烟后** | 即时+5 | 短暂放松 |
| **运动后** | 即时+10 | 内啡肽释放 |
| **Merritt夸奖** | +15 | 被认可 |
| **Merritt批评** | -15 | 被否定 |

**心情也影响其他系统：**
- 心情好 → 疲劳消耗-20%（心情好干活不累）
- 心情好 → 抽烟频率-30%（不需要烟陪伴）
- 心情差 → 疲劳消耗+30%（情绪低落耗能高）
- 心情差 → 抽烟频率+50%（需要烟陪伴）
