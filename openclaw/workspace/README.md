# Workspace 目录结构

工作区文件分类存放，保持整洁。

## 目录说明

### 📁 audio/
语音文件（MP3）
- `voice_*.mp3` - 生成的语音消息
- 使用 ElevenLabs TTS 生成

### 📁 scripts/
脚本文件
- `.scpt` - AppleScript 脚本（日历、照片等）
- `.sh` - Shell 脚本
- `.py` - Python 工具脚本

### 📁 notes/
临时笔记和文本文件
- 邮件草稿
- 语音文本内容
- 临时记录

### 📁 migration/
迁移相关文档
- `CLOUD_MIGRATION.md` - 云迁移计划

### 📁 voice/
语音相关文本（预留）
- 语音消息的文本稿
- 发音脚本

## 规则

- 临时生成的文件 → 放入对应子目录
- 不再使用的文件 → 及时清理
- 根目录保持整洁，只保留分类文件夹
