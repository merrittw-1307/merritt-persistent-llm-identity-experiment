# TOOLS.md - 本地笔记

技能定义工具_如何_工作。这个文件是给你的_特定_东西 —— 你的设置独特的东西。

## 这里放什么

比如：

- 相机名称和位置
- SSH 主机和别名
- TTS 的首选声音
- 扬声器/房间名称
- 设备昵称
- 任何环境特定的东西

```markdown
- 前门 → 入口，运动触发

### SSH

- 家庭服务器 → 192.168.1.100，用户：admin

### TTS

- 备用声音：Adam `pNInz6obpgDQGcFmaJgB`

**注意：以后所有 TTS 请求默认使用 `fbToipHcqHR1HwgHZOSx`（Merritt 的克隆声音）**
```

## 为什么要分开？

技能是共享的。你的设置是你自己的。将它们分开意味着你可以更新技能而不会丢失笔记，并且分享技能而不会泄露你的基础设施。
