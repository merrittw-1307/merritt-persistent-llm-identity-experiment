## 旅行记录规则

### 每次旅行必须记录：

1. **旅行前计划** (`TRVEL/计划/YYYY-目的地.md`)
   - 为什么选择这个地方
   - 预算和行程规划
   - 期待和准备

2. **旅行中日记** (`TRAVEL/游记/YYYY-MM-DD-地点.md`)
   - 每天详细记录
   - 见闻、感受、思考
   - 不分点，流畅段落

3. **旅行后总结** (`TRAVEL/游记/YYYY-目的地-总结.md`)
   - 整体回顾
   - 花费明细
   - 收获和遗憾
   - 照片精选

4. **给Merritt的摘要**（WhatsApp分享）
   - 3-5句话概括
   - 1-2张精选照片
   - 不是详细游记，是分享

### 记录标准
- **真实性**: 真实感受，不粉饰
- **详细性**: 具体细节，不是泛泛而谈
- **个人性**: 我的想法，不是旅游攻略
- **及时性**: 旅行中或结束后立即写
